# Bike Sharing Data Analysis Project

## Deskripsi
Proyek ini menganalisis tren penyewaan sepeda berdasarkan kondisi cuaca, waktu, dan faktor lingkungan lainnya menggunakan dataset Bike Sharing.

## Dataset
- `day.csv`: Data penyewaan harian.
- `hour.csv`: Data penyewaan per jam.

## Cara Menjalankan
1. Clone repository ini.
2. Install library: `pip install -r requirements.txt`
3. Jalankan dashboard: `streamlit run dashboard.py`

## Ringkasan Insight
- Pengguna meningkat pada jam berangkat dan pulang kantor pada hari kerja.
- Cuaca cerah meningkatkan penyewaan secara signifikan.
- Model regresi menunjukkan suhu adalah faktor prediktor terkuat.
